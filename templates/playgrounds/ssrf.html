What is SSRF?

Server-Side Request Forgery forces the application to make HTTP requests to attacker-controlled destinations — including internal network addresses not accessible from the internet.

Common targets: cloud metadata (AWS IMDSv1), internal APIs, Redis, Elasticsearch, Spring Boot Actuator.
